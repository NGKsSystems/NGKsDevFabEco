================================================================================
PREREQUISITES: What You Need Before Running DevFabEco
================================================================================

REQUIRED (these will be validated automatically):

1. PYTHON 3.11 or later
   ✓ Check: python --version
   ✓ Windows: https://www.python.org/downloads/ (add to PATH)
   ✓ Via Chocolatey: choco install python

2. VISUAL STUDIO or MSVC Build Tools
   (For C++ compilation. Skip if your project doesn't use C++.)
   ✓ Professional/Community/BuildTools edition all work
   ✓ Must include "Desktop Development with C++"
   ✓ Download: https://visualstudio.microsoft.com/downloads/

3. GIT
   ✓ Check: git --version
   ✓ Download: https://git-scm.com/download/win
   ✓ Via Chocolatey: choco install git

4. POWERSHELL 5.0 or later (or PowerShell Core 7+)
   ✓ Check: $PSVersionTable.PSVersion
   ✓ Included in Windows 10/11
   ✓ For older Windows: https://docs.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-windows

CONDITIONAL (depend on your project):

5. Qt 5.15+ or Qt 6.0+ (only if your project uses Qt)
   ✓ Download: https://www.qt.io/download-open-source
   ✓ Set environment variable: QtDir=C:\Qt\6.0\msvc2019_64
   ✓ Via Chocolatey: choco install qt

VALIDATION:

Run the automatic prerequisite checker:
  .\tools\check_prereqs.ps1

This script will:
  ✓ Verify Python version
  ✓ Verify MSVC environment
  ✓ Verify git
  ✓ Report any missing components
  ✓ Provide fix guidance

DO NOT PROCEED until all checks pass.

WINDOWS ENVIRONMENT:

For MSVC to work automatically:
  ✓ VS Enterprise, Professional, or Community (any edition with C++ support)
  ✓ Build Tools for Visual Studio also works
  ✓ Visual Studio Code + command-line MSVC works
  ✓ The MSVC compiler (cl.exe) must be on PATH

If MSVC is not found:
  1. Install Visual Studio with "Desktop Development with C++"
  2. OR install Build Tools for Visual Studio
  3. Verify by running: cl.exe /?
  4. Re-run the bundle

TROUBLESHOT INSTALLATION:

Python on Windows PATH:
  Check: python --version
  Fix: Add C:\Users\<YOU>\AppData\Local\Programs\Python\Python311 to PATH
       (restart PowerShell after changing PATH)

MSVC Not Found:
  Check: cl.exe /?
  Fix: Install Visual Studio with C++ support (see above)
       After install, restart PowerShell

Git Not Found:
  Check: git --version
  Fix: https://git-scm.com/download/win
       Restart PowerShell after install

QUICK VALIDATION SCRIPT:

Copy and run this in PowerShell to check everything:

  python --version  # Should be 3.11+
  git --version
  pwsh -Version  # PowerShell
  cl.exe /?      # MSVC (optional, C++ projects only)

If all show versions, you're ready!

NEXT STEP:
Run: .\run.ps1
